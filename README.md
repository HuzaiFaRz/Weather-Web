## Overview
A weather application that fetches real-time data from the OpenWeather API. Users can search by city or use their current location to get live weather updates.

## Features

- Real-time weather data display
- City search functionality
- Geolocation-based weather
- Dynamic UI updates based on weather
- Responsive layout
